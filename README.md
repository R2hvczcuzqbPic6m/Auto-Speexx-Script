# Auto-Speexx-Script

**开箱即用，解放双手。**

自动完成 Speexx 平台视频播放、练习答题、发音练习跳过、跨页自动续跑，可设定参数模拟人工做题时长。

## 安装

1. 前置：浏览器安装油猴插件（Tampermonkey/Violentmonkey）；
2. 方式一：点击 [GreasyFork 页面](https://greasyfork.org/en/scripts/571011-autospeexx) 的「Install」按钮，确认安装；（国内镜像：[🔗GFork 镜像](https://gfork.zh-tw.eu.org/zh-CN/scripts/571011-autospeexx) [🔗GFork Proxy 镜像](https://home.greasyfork.org.cn/zh-hans/info/#/zh-CN/scripts/571011/detail)）
4. 方式二：从 [Github](https://github.com/R2hvczcuzqbPic6m/Auto-Speexx-Script) 下载后手动导入；
5. 刷新 Speexx 页面，即可看到脚本面板。

## 使用说明

1. 进入 [Speexx](https://portal.speexx.cn/) 并打开练习页面；
2. 页面右侧会显示「AutoSpeexx」面板（可最小化），点击神秘绿色按钮即可开始自动化；
3. 「参数配置」页可自定义配置学习时长等。

## 注意事项

1. 主页 / 非练习页面启动无效；
2. 建议设为和自己平常差不多的完成时长，请勿过度缩短部分配置项避免数据被平台判定异常；
3. 请勿在同一页面反复来回点击启动和停止按钮，会变得不幸；
4. 免责声明：前面忘了中间忘了总之脚本仅用于辅助学习。
